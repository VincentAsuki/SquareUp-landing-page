# SquareUp-landing-page
A modern digital product studio landing page built with HTML and Tailwind CSS featuring a futuristic dark UI, neon-green accents, service sections, testimonials, FAQ, and contact form.



## 🚀 Features

- Modern dark-themed UI
- Built with HTML & Tailwind CSS
- Hero section with CTA buttons
- Services section
- Why Choose Us section
- Client testimonials
- FAQ accordion layout
- Contact form section
- Footer with navigation & socials
- Smooth hover effects and transitions

---

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS
